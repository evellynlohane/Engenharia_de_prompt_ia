# Engenharia_de_prompt_ia
College 

Este repositório será dedicado à disciplina de Engenharia de Prompt e Aplicações em Inteligência Artificial. Tem como objetivo reunir conteúdos, exemplos práticos e experimentos relacionados à criação de prompts eficientes para modelos de linguagem (LLMs), explorando técnicas de estruturação, contexto, instruções e refinamento de prompts para obter respostas mais precisas e úteis.
Além disso, o repositório busca demonstrar na prática como a engenharia de prompt pode ser aplicada em diferentes cenários de IA, contribuindo para o desenvolvimento de soluções mais inteligentes, eficientes e alinhadas às necessidades dos usuários.
