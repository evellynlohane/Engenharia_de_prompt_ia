# Engenharia_de_prompt_ia
College 

Engenharia de prompt é a disciplina que estuda como escrever entradas (prompts) claras, estruturadas e contextuais para orientar modelos de IA, especialmente modelos de linguagem grandes (LLMs), a gerar saídas de alta qualidade.
Ela combina arte e ciência: você experimenta com formato, contexto, exemplos e restrições no texto do prompt, observa a resposta da IA e vai iterando até chegar no comportamento desejado.
